## 五、如何导入

### 方法一：单个主文件导入
```sql
source H:\asocial\social-retail\backend\src\main\resources\db_new\social_retail_db.sql
```

### 方法二：逐个文件导入（按顺序）
1. 先导入 `social_retail_db.sql`（主文件，包含了所有内容）
2. 或按外键依赖顺序逐个导入所有独立文件

- 先执行 schema.sql 创建基础表结构，再执行 db_new 的数据文件
- 或者直接使用 db_new 中的文件（包含了 DROP TABLE + CREATE TABLE + INSERT）
